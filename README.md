# Diabetes Prediction Through AI

## Project Overview
This project is an AI-powered Diabetes Prediction System developed using Machine Learning. The system predicts whether a person is diabetic or non-diabetic based on medical data such as glucose level, BMI, insulin, blood pressure, and age.

The main aim of this project is to use Artificial Intelligence in healthcare for early diabetes prediction.

---

## Objectives
- Build a machine learning model for diabetes prediction
- Analyze medical data
- Train and test prediction models
- Provide accurate prediction results
- Understand AI applications in healthcare

## Dataset Features
The dataset contains the following features:

- Pregnancies
- Glucose
- BloodPressure
- SkinThickness
- Insulin
- BMI
- DiabetesPedigreeFunction
- Age

### Target Variable
- 0 = Non-Diabetic
- 1 = Diabetic
